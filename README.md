# README

アプリケーション名：Otarue

アプリケーション概要：北海道小樽市の観光サイトを体に作成。

URL：https://otarue-33947.herokuapp.com/

利用方法：WEBデザイン主体のHP。ユーザー側でアクション実行可能なページは購入機能・問合せ機能の２点。

目指した課題解決：小樽市への観光を検討している人へ、どのような経路で巡っていけば良いかを悩む、という課題を解決する為に作成。

要件：購入機能/商品の購入ができる
